Project

Programming language

My project is to develop a game within python using pygame and pygame zero. Both game engines are similar, however, pygame zero is better for beginners to learn as they can program without writing an event loop

Game explanation

The game that I created was inspired by the basic collecting objects game which records the number of time the player collects each object as the score. My game includes the character, a puppy, collecting shells on the beach. Every time the puppy touches the shell the score increases by one and the shell appears in a new random position. The only way to stop the game is for the timer of 30 seconds to run out or to press the 'x' key. Once the timer runs out, a 'game over' screen will appear along with the score the player earned. If the score is greater than previous score then a 'congradulations' message would appear as well.
