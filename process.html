I then started creating the game by following the tutorial on starting a pygame zero file and declared variables such as my character image, shell image and background image. I also programmed my character to be able to move following the arrow keys.

Displaying everthing on screen

To display all images and text on screen I used the built in "draw" function. To make the program stop the game mode and draw the "game over" screen, I used "if timer = 0" and then "draw". I then cleared the screen to make the program seem like the screen is new. To show the text on screen I use the "screen.draw.text(Score..." function and added the end score as a string. I had to round the time and used the same function to display it. 

                  def draw():
                     
                     if timer <= 0:
                        screen.clear()
                        screen.draw.text('Game Over', (420,270), color=(255,255,255), fontname="vc.ttf", fontsize=40)

                        screen.draw.text('Score: ' + str(score), (420,330), color=(255,255,255), fontname="vc.ttf", fontsize=40)

                     else:
                        game_bg.draw()
                        shell.draw()
                        char.draw() 
                        screen.draw.text('Score: ' + str(score), (15,10), color=(255,255,255), fontname="vc.ttf", fontsize=20)
                        screen.draw.text('Time: ' + str(round(timer)), (330,10), color=(255,255,255), fontname="vc.ttf", fontsize=20)

Keeping high score

To keep a high score I needed the python to call a text file with the high score within and read the score. After, python would compare the score of the player now and the previous high score. If the current score is higher then python would write the current score over the high score in the text file. 

                  score_now = int(str(score))
                   
                  # gets the high score of the game
                  def get_high_score():
                     f = open('score.txt', 'r') #opens file in read mode
                     file = f.readlines()
                     high_score = int(file[0])
                     f.close()

                     return high_score


                  # saves high score
                  def updateFile():
                     f = open('score.txt', 'r') #opens file in read mode
                     file = f.readlines()
                     last = int(file[0])
                     
                     if last < int(score): #sees if current score is higher than the last
                        f.close()
                        file = open('score.txt', 'w') #opens th efile and rewrites the new score
                        file.write(str(score))
                        file.close()

                        return score
                     return last

                  high_score = int(get_high_score())
